# ゼロのはいったTensor(行列)を作る

## Sample(2.7)

![](/img/np_zerotensor.png)

```python
import numpy as np

mat = np.zeros([3,2])
print mat
```

結果
```shell
[[ 0.  0.]
 [ 0.  0.]
 [ 0.  0.]]
```
